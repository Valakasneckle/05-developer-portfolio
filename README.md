## Project Overview

Developer Portfolio 2.0 is an advanced personal website for presenting full-stack engineering skills, AI projects, commercial web platforms, case studies, architecture thinking, and freelance availability.

The site is structured for technical credibility: project case studies, service positioning, skills grouped by category, architecture showcase, experience timeline, and a professional contact flow.

## Business Goal

The website helps potential clients, recruiters, and collaborators quickly understand the developer's skills, project quality, technical range, and suitability for freelance or contract work.

It supports outreach on GitHub, LinkedIn, and direct client conversations by making project depth and service offerings immediately visible.

## Key Features

- Advanced developer portfolio homepage
- Projects page with full project grid
- Dynamic project detail pages (`/projects/[slug]`)
- Skills grouped by category (Frontend, Backend, AI, DevOps, Databases, Architecture, UI/UX)
- Services section for freelance clients
- Architecture showcase
- Experience / timeline section
- Blog preview
- Contact / lead generation page
- Responsive desktop and mobile layout
- Reusable component architecture
- Professional personal brand design
- SEO-friendly page structure

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm

### Installation

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
```
