# Features

## Portfolio Features

- Home page with full section flow
- Projects page listing all case studies
- Dynamic project detail pages
- About page with bio, skills, and experience
- Blog page with article previews
- Contact page with lead form

## Project Presentation Features

- Project cards with category, year, and tech stack
- Project detail with problem, solution, features, architecture notes, and business value
- GitHub and live demo links per project
- Future improvements section on each project

## Service Positioning Features

- Seven freelance service cards
- Deliverables listed per service
- Services section on homepage for client outreach

## Contact Features

- Name, email, company fields
- Project type selector (Landing Page, SaaS Dashboard, E-commerce, AI Integration, etc.)
- Budget range selector ($1k–$3k through $10k+)
- Message field
- UI-only submission with success state

## UI Features

- Dark professional theme with purple/blue accents
- Responsive mobile-first layout
- Framer Motion animations on hero, skills tabs, testimonials
- Reusable Button, Card, Badge, Section, Input, Textarea, Tabs components
- Consistent spacing and typography

## Developer Features

- TypeScript throughout
- Static data files in `data/`
- App Router with static generation for project pages
- `cn()` utility for class merging
- `.env.example` for site URL configuration
- Documentation in `docs/` and `architecture/`
